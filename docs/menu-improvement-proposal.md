## 背景・課題認識（提案時の記録）

### 他のAIとの相談内容から抽出した課題
1. **ISSN表示位置の最適化**：フッター vs サイドバー vs 専用ページ
2. **学術的信頼性の確保**：専門雑誌としての体裁を整える必要性
3. **デザインのシンプルさとの両立**：情報過多を避けつつ必要な情報を提供
4. **既存のStarlightベース構造の活用**：現在の清潔感のあるデザインを維持

### 現在のメニュー構造の問題点
- 「記事を読む」配下がバックナンバーのみで不十分
- 投稿関連機能が未完成（コメントアウト状態）
- 学術雑誌として必要なメタデータ（ISSN等）の表示が不明確
- 最新号への直接アクセスルートが不明

## ISSN表示の具体的提案

### 最終推奨案：サイドバー下部配置

#### 理由
1. **常時表示**: 全ページでアクセス可能
2. **視覚的バランス**: フッターほど重くならない
3. **学術的信頼性**: 必要な情報を適切に提示
4. **モバイル対応**: ハンバーガーメニュー内でも確認可能

#### 実装詳細
```astro
---
// Starlight カスタムコンポーネント
// src/components/SidebarFooter.astro
---
<div class="sidebar-publication-info">
  <div class="border-t border-gray-200 dark:border-gray-700 pt-4 mt-6">
    <div class="text-xs text-gray-600 dark:text-gray-400 space-y-1">
      <p class="font-medium text-gray-700 dark:text-gray-300">刊行物情報</p>
      <p>ISSN: 2188-0972 (Online)</p>
      <p>発行：ニッチェ・ライフ編集委員会</p>
      <a 
        href="https://www.jstage.jst.go.jp/browse/nichelife/" 
        target="_blank"
        class="text-blue-600 hover:text-blue-800 dark:text-blue-400"
      >
        J-STAGEで読む ↗
      </a>
    </div>
  </div>
</div>
```

### 代替案：フッターからの完全削除

他のAIが提案していた「フッターからの完全削除」も検討に値します：

**フッター簡素化後**:
```html
<footer class="text-center py-4 text-gray-600">
  <p>© 2024 ニッチェ・ライフ編集委員会</p>
</footer>
```

**メリット**:
- 究極にシンプルなフッター
- ページの最下部が軽やかになる
- サイドバーに情報を集約
