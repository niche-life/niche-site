# プロジェクト概要書

## サイトの目的とミッション

### 基本情報
- **サイト名**: ニッチェ・ライフ（Niche Life）
- **URL**: https://www.niche-life.com
- **性質**: 無料公開ウェブ生物雑誌
- **対象**: 生物学研究者、自然愛好家、学生、一般市民

### ミッション
「生物多様性の基礎記録を学術的に蓄積し、広く一般に公開することで、ネイチャーポジティブ（生物多様性の回復）に貢献する」

## ターゲットユーザー

### プライマリーユーザー
1. **研究者・専門家**
   - 生物学研究者、博士課程学生
   - 自然史博物館学芸員
   - 環境調査会社の研究員

2. **アマチュア自然愛好家**
   - 自然観察サークル会員
   - フィールドワーク愛好者
   - 写真撮影愛好者

3. **教育関係者**
   - 高校・大学教員
   - 環境教育指導者
   - 理科教諭

### セカンダリーユーザー
- 環境政策立案者
- ジャーナリスト
- 一般市民（生物に興味がある層）

## サイトの価値提案

### ユニークバリュー
1. **投稿無料・全文無料公開**: 誰でもアクセス可能
2. **学術的引用可能**: DOI付与により研究に活用可能
3. **多様性受容**: 専門論文からフィールドレポートまで
4. **永続的保存**: J-STAGEとの連携により長期保存

### 既存サービスとの差別化
- 学会誌: 会員制限なし、投稿ハードルが低い
- SNS投稿: 学術的引用可能、検索・分類が容易
- 一般ブログ: 査読体制、永続性の担保

## 理想的なリンク構造

### 情報アーキテクチャ
```
トップページ
├── 雑誌について
│   ├── ニッチェライフとは → ミッション・目的
│   ├── 運営体制 → 編集委員一覧・プロフィール
│   └── 新着ニュース → 発行情報・更新情報
├── 記事を読む
│   ├── バックナンバー → 全巻号のブラウジング
│   └── [個別巻号] → 記事一覧・PDF・DOIリンク
├── 記事を投稿する
│   ├── 投稿規定 → ルール・フォーマット
│   ├── 投稿の手引き → 具体的な書き方
│   ├── 投稿フォーム → オンライン投稿
│   └── よくある質問 → FAQ・サポート
└── 外部連携
    ├── J-STAGE → 正式版・引用情報
    └── 問い合わせ → 編集委員会連絡先
```

### SEO・導線設計

#### 検索流入の想定
1. **学名検索**: 「○○ ○○ 分布」「△△ 観察記録」
2. **研究テーマ**: 「生物多様性 調査」「フィールドワーク」
3. **投稿意図**: 「学術雑誌 投稿無料」「生物 記録 発表」

#### 内部リンク戦略
- 巻号詳細ページ → 類似テーマの他巻号
- 記事ページ → 著者の他の寄稿
- 編集委員紹介 → 担当分野の記事一覧
- 投稿規定 → 実際の記事例・参考事例

### ユーザージャーニー

#### 新規読者
1. 検索エンジン → 特定記事
2. 記事閲覧 → 関連記事・同じ巻の他記事
3. バックナンバー探索
4. 最新号への関心 → ニュース購読

#### 投稿検討者
1. 投稿規定確認
2. 既刊記事のレベル・形式確認
3. 投稿手引き詳読
4. 編集委員会への相談
5. 投稿フォーム利用

#### リピーター
1. トップページ → 新着情報確認
2. 最新号チェック
3. 特定分野の記事検索
4. 研究・教育での活用

## 技術的要件とパフォーマンス目標

### 必須機能
- レスポンシブデザイン（スマホ・タブレット対応）
- 高速表示（LCP 2.5秒以下）
- アクセシビリティ対応（WCAG 2.1 AA準拠）
- SEO最適化（構造化データ対応）

### 運用要件
- 編集委員による簡単な更新作業
- 新巻号の迅速な公開（入稿から1週間以内）
- バックアップ・災害対策
- セキュリティ・プライバシー保護
