## ユーザージャーニーマップ

### 1. 研究者（論文・記録を読みたい）
```
検索エンジン → 特定記事 → 記事詳細・DOI → 引用・ダウンロード
             ↓
       関連記事・同じ巻 → バックナンバー探索 → 定期購読検討
```

### 2. 投稿検討者（記録を発表したい）
```
トップページ → 投稿案内 → 投稿規定確認 → 既刊記事レベル確認
            ↓
     投稿の手引き → 原稿作成 → オンライン投稿 → 査読・掲載
```

### 3. 一般読者（生き物に興味がある）
```
SNS・リンク → 特定記事 → 読みやすい記事・コラム → 他の記事
            ↓
      雑誌について → ニッチェライフの目的 → 投稿検討・応援
```

### 4. 編集関係者（更新作業）
```
管理画面（想定） → 新巻号データ投入 → プレビュー → 公開
                ↓
         ニュース投稿 → SNS連携 → 読者反応確認
```

## メニュー構造の改善提案

### 現在の課題
1. **「記事を読む」が不十分**：バックナンバーのみで最新号へのアクセスが不明確
2. **投稿機能が不完全**：手引きやフォームがコメントアウト
3. **学術情報の不足**：ISSN等のメタデータが配置されていない
4. **検索・発見機能なし**：過去記事の発見性が低い
